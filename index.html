scripts

organizer.py
sorts files into folders automatically by file type
asks which folder to organize, then sorts every file into subfolders like Music/, Images/, Documents/ etc based on extension. supports mp3, jpg, pdf, zip and more. built with os and shutil.
randompick.py
picks random songs from my music library + plays them
walks through all subfolders recursively, finds every mp3, lets you pick how many songs you want, then plays them one by one using mpg123. built with os, random, and subprocess.
moodpicker.py
recommends music based on how im feeling
maps moods to specific music folders — happy goes to Dakota's MP3, sad goes to Ethel Cain, angry goes to Frightened Little Asshole, chill goes to Run rabbit run. picks a random song from the matching folder.
lyrics.py
fetches lyrics from genius api
uses the lyricsgenius library to hit the genius api. type in an artist and song title and it fetches and prints the full lyrics in your terminal. requires a genius api key.
facts.py
pulls random facts from the internet
hits the uselessfacts api with no api key needed. uses urllib and json which are built into python. run it multiple times for different facts every time.
music_backup.sh
logs my entire music library every night at midnight
bash script scheduled via cron to run at midnight every day. recursively lists every file in ~/Music and appends it to music_log.txt with a timestamp.
auto_organizer.py
auto-organizes downloads every monday at 9am
same as organizer.py but with the folder hardcoded to ~/Downloads so it can run automatically via cron without needing user input. scheduled every monday at 9am.
